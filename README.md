## Installation and Setup
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/techiuv/news-hunt.git
   cd news-hunt
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

5. **Preview the Production Build**:
   ```bash
   npm run preview
   ```
